# Waiveo Next

An open platform for digital signage, home automation, and device control —
a single self-hosted box that drives your screens and devices, built
contracts-first.

**Status:** early. The versioned contracts in [`contracts/`](contracts/) are
the product; implementations land behind them. Nothing here is ready to run
yet.

## Repository layout

- `contracts/` — the versioned protocol and schema contracts (the source of truth)
- `conformance/` — corpora, validators, and drivers that prove an implementation honest
- `scripts/` — repo tooling
- `docs/` — published documentation
